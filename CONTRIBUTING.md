# Contributing to IPL Analytics Dashboard

Thank you for your interest in contributing! 🏏

## Ways to Contribute

- 🐛 **Report bugs** via [Issues](../../issues/new?template=bug_report.md)
- 💡 **Request features** via [Issues](../../issues/new?template=feature_request.md)
- 🔧 **Submit a pull request** with improvements
- ⭐ **Star the repo** to help others discover it

## Pull Request Guidelines

1. **Fork** the repository
2. Create a **feature branch**: `git checkout -b feature/your-feature-name`
3. Make your changes with **clear, descriptive commits**
4. **Test** in Chrome, Firefox, and Safari before submitting
5. Open a PR with a description of what you changed and why

## Commit Message Format

```
Add: new chart for player strike rates
Fix: venue chart not rendering on Firefox
Update: README with new screenshots
Refactor: clean up Plotly config in Page 2
```

## Code Style

- Keep all JS inline within the HTML file it belongs to
- Use the existing CSS variable system (`--orange`, `--navy-card`, etc.)
- Match the existing dark theme aesthetic
- Add comments for any non-obvious chart configurations

## Data Contributions

If you have updated or extended IPL data:
- Document the source clearly
- Note any columns added or changed
- Update `docs/DATA_SOURCES.md`

## Questions?

Open an issue with the `question` label.
